Vacation Planner Application

The Vacation Planner Application is an Android mobile app developed in Java using Android Studio and the Room persistence framework. It allows users to create, manage, and organize vacations and excursions with alert and reporting capabilities.

Features

  • Create, edit, and delete vacations
  • Add and manage excursions
  • Date validation and alert notifications
  • Share vacation summaries
  • Search and generate vacation reports
